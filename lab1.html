Lab 1

    git commands
  • git clone "your repo link" - командыг ашиглан та тухайн репозиторио бүрэн хуулбарлаж, таны компьютер дээр шинэ директор үүсгэнэ.
  • git add . - Энэ командыг ашиглахад таны ажлын директор дахь бүх өөрчлөлтүүд stage буюу бэлтгэсэн төлөвт орно.
  • git commit -m "your message" - командыг ашиглан та staged буюу бэлтгэсэн өөрчлөлтүүдийг хадгалж, нэгтгэх болно. -m "" нь коммитын тайлбарыг тодорхойлдог. Хавсаргасан сэдвийн дотор өөрчлөлтүүдийн талаарх тайлбарыг бичнэ
  • git push origin main - командыг ашиглан таны локал буюу компьютер дээрх Git репозиторио, өөрчлөлтүүдийг холбогдох сервер буюу remote репозитори руу илгээнэ.
